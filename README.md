✨ About the Project

NCR Healthy Energy is a modern e-commerce web application designed to showcase and present healthy energy products with a clean and engaging user interface.

The application focuses on delivering a smooth browsing experience with structured product listings, responsive design, and interactive UI components.

🚀 Features

🛍️ Product listing and category-based display

🎯 Product details and quick-view sections

💬 UI sections like testimonials and promotional highlights

📱 Fully responsive design (mobile-first approach)

⚡ Fast and optimized frontend performance

🎨 Clean, modern, and user-friendly interface

🧩 Reusable and modular component-based architecture

🛠️ Tech Stack

Frontend: React.js, Vite, CSS Modules  
UI/UX: Flexbox, CSS Grid, Responsive Design  
Deployment: Vercel
